MetaSearch Static Package

This is a standalone static version of the MetaSearch application that can be deployed to any web server or static hosting service.

Download ZIP Package View Demo

What's Included

  • Static HTML file with all necessary code
  • Detailed README with deployment instructions
  • CSS file for offline use
  • Offline-friendly version with external CSS

Quick Deployment

After downloading, you can deploy this package to:

  • GitHub Pages
  • Netlify, Vercel, or other static hosting services
  • Any web server that can serve static files

Using the Application

To use MetaSearch, you'll need to:

  1. Deploy the static files to your chosen hosting platform
  2. Get a SerpAPI key from serpapi.com
  3. Enter your API key in the application's "API Settings" screen
  4. Start searching across multiple engines!

Manual Installation

If you prefer, you can also use these individual files:

  • index.html - The main application file
  • offline.html - Version with external CSS
  • styles.css - External CSS file
  • README.md - Documentation
